# My APP Documents Feature

## Overview

The Documents feature in My APP allows users to manage and access various types of documents conveniently in one place. This README provides an overview of the feature and guides users on how to use it on both desktop/tablet and mobile devices.

## Getting Started

### Desktop/Tablet

1. Click on the "Documents" option in the hamburger menu.
2. The Document Dashboard will load with the "Joining Documents" tab selected by default.
3. Explore other tabs by clicking on them.
4. View a document by clicking the View icon on the document row.
5. When no joining documents are available, the message "No Joining Documents available to display" will be shown.

### Mobile

1. Click on the "Documents" option in the navigator menu.
2. The Document Page will load with a list of menu options.
3. Click on a menu option to view the respective documents.
4. Each menu option has a clickable chevron for additional actions.
